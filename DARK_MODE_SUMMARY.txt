                           QUICK REFERENCE
================================================================================

Toggle Button:
  Location: Top-right corner of header
  Visual: Moon icon (light mode) / Sun icon (dark mode)
  Behavior: Click to toggle, remembers choice

Dark Mode Colors:
  Background: #1a1a1a (very dark, not pure black)
  Text: #f5f5f5 (off-white, not pure white)
  Borders: #333333 (dark gray)
  Why: Reduces eye strain, easier to read

How It Works:
  1st Visit: Auto-detects OS preference
  Toggle: User saves preference to localStorage
  Return: Uses saved preference (ignores OS)

Keyboard: Tab to button, Space/Enter to toggle
Screen Reader: Announces "Toggle dark mode, button"

================================================================================
                           TROUBLESHOOTING
================================================================================

Problem: Dark mode not working
  → Hard refresh (Cmd+Shift+R)
  → Check console (F12) for errors
  → Verify dark-mode.js file exists

Problem: Button not visible
  → Hard refresh
  → Check style-optimized.css is linked (not style.css)
  → Check .theme-toggle CSS exists

Problem: Colors look wrong
  → Check @media (prefers-color-scheme: dark) in CSS
  → Verify color variables are correct
  → Hard refresh to clear cache

Problem: Theme doesn't persist
  → Check localStorage enabled in browser
  → F12 → Application → Local Storage
  → Look for "agust-islandia-theme" key

For more troubleshooting, see DARK_MODE_TESTING.md
